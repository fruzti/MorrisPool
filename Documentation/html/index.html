Bienvenido a la Documentación de MorrisPool!¶

Morris Pool es un proyecto basado en Python para el Lab. de Psicobiología del Centro de Investigación Biomédica de Occidente. El proyecto ha sido sido desarrollado desde sus bases hasta necesidades específicas debido a requerimientos de la investigación y restricciones del experimento. Además, este proyecto es parte de un paquete de software para experimentos de comportamiento utilizando ratones desarrollado por el mismo autor.

El siguiente material es la documentación del proyecto, conteniendo todos los métodos, las clases y la estructura del código disponible para desarrollo futuro del proyecto y mantenimiento de software.

El proyecto fue realizado bajo la supervisión de:
  • Laboratorio de Psicobiología

    Centro de Investigación Biomédica de Occidente

Responsable:
  • Dr. Ignacio Gonzales Burgos
